# 🎓 Student Management System (Java)

A console-based Student Management System developed using Java that performs CRUD operations and automatic grade calculation.

## 📌 Features
- ➕ Add Student
- 📋 View All Students
- 🔍 Search Student by ID
- ❌ Delete Student
- 🏆 Automatic Grade Calculation

## 🛠 Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner Class

## 🎯 Project Objective
This project was developed to understand:
- Java fundamentals
- OOP concepts
- CRUD operations
- Data handling using ArrayList
